1. Apa itu CSS Framework?
- Suatu standart CSS yang dibuat untuk developer agar memudahkan dalam mendesain tampilan website

2. Apakah bootstrap satu-satunya CSS Framework yang dapat kita gunakan?
- Tidak, ada beberapa CSS Framework lainnya

3. Sebutkan beberapa CSS Framework selain Bootstrap
- Tailwindcss, Sass, Foundation, Susy, Bulma, dll

4. Bagaimana tata cara penggunaan Bootstrap?
- Kita bisa menginstallnya di komputer local atau bisa memaanggil link cdn pada html kita. Kemudian untuk stylingnya
bisa menggunakan class pada html sesuai dengan dokumentasi pada Bootstrap
